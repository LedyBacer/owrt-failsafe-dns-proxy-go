# Project status

**English** | [Русский](../ru/project-status.md)

Daemon package version: `0.2.1-r1`.

The main scenario has been verified on a Xiaomi Redmi Router AX6S with
OpenWrt 24.10.7: installation, LuCI, primary upstream failure, fallback,
recovery, failback, reload, dnsmasq rollback, and package lifecycle. A
multi-day soak test is ongoing.

Builds for other platforms are verified by compilation and package smoke tests,
not by real hardware. The detailed readiness ledger is in
[PLAN.md](../PLAN.md).
